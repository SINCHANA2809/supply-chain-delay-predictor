# Supply Chain Delay Predictor

A machine learning pipeline that predicts whether an e-commerce shipment will be delayed.

## Setup Instructions
1. Clone this repo
2. Create a virtual environment: `python -m venv venv`
3. Activate it: `venv\Scripts\activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Run the notebook: open `notebook.ipynb` in VSCode

## Dataset
E-Commerce Shipping Dataset from Kaggle:
https://www.kaggle.com/datasets/prachi13/customer-analytics

## Project Structure
- `notebook.ipynb` — EDA and model training
- `main.py` — FastAPI app (coming soon)
- `data/` — dataset folder (not pushed to GitHub)
- `requirements.txt` — all dependencies
